# Support

## Where To Start

If you are trying to use the project:

- start with [README.md](README.md)
- then read [docs/install.md](docs/install.md)
- then read [docs/usage.md](docs/usage.md)

## Open An Issue When

- the installer fails
- the installed skill is not recognized
- an example or integration does not build
- documentation is wrong or incomplete

Use the issue templates in `.github/ISSUE_TEMPLATE/`.

## Open A Discussion When

- you want help choosing the right integration path
- you want feedback on a new use case
- you want to propose a bigger direction before opening a PR

## Security

For vulnerabilities, use [SECURITY.md](SECURITY.md) instead of a public bug report.
