# Design Rules Reference

Use this file when you need the condensed AI-facing version of the visual rules.

## Must Keep

- Strong typography
- Intentional palette
- Visible composition
- Text-linked motion
- Clear mode selection

## Must Avoid

- Generic SaaS template aesthetics
- Placeholder-level animation
- Default purple-on-white gradients
- Weak type hierarchy
- DOM text measurement in the relayout path

## Preferred Aesthetics

- editorial
- modern poster
- kinetic type
- compositional asymmetry
- atmospheric but readable surfaces

## Motion Heuristic

If removing the motion would leave the same core idea, the motion is too weak.

## Typography Heuristic

If the page still works with default browser fonts and loses nothing, the type system is too generic.
