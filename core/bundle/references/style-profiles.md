# Style Profiles

Use these profiles to keep the bundle from collapsing into one default visual language.

Choose one primary profile for a result. Mix only when the prompt explicitly asks for a hybrid.

## `editorial-paper`

Use when the brief is literary, curatorial, manifesto-like, or exhibition oriented.

- warm paper or muted neutral background
- serif-led headline or body hierarchy
- asymmetrical spread composition
- routed text, pull quotes, measured blocks, calmer motion

Prompt words that fit:

- editorial
- literary
- exhibition catalogue
- essay spread
- paper texture

## `technical-lab-white`

Use when the brief should feel sparse, technical, algorithmic, or similar to `algo.qq.com/#intro`.

- near-white or slightly tinted white field
- light technical sans-serif
- micro-labels or signal tags distributed across the page
- central measured word, glyph cloud, or signal aggregation
- restrained color accents, usually violet, blue, magenta, or graphite
- motion as drift, aggregation, dispersion, or local field response

Prompt words that fit:

- technical lab
- algorithm field
- signal map
- sparse white interface
- computational typography

## `kinetic-dark-poster`

Use when the page should feel cinematic, higher-contrast, and motion-forward.

- dark field
- stronger glow or contrast edges
- louder display type
- glyph or particle motion doing more of the visual work

Prompt words that fit:

- poster
- cinematic
- dark field
- luminous type
- motion graphic

## `compact-measured-ui`

Use when the request is practical but still should avoid a generic dashboard look.

- tight modules
- smaller shells
- utility-first hierarchy
- measured text blocks carrying layout stability

Prompt words that fit:

- bubbles
- card wall
- accordion
- compact feed
- dense messaging surface

## Anti-pattern

If the profile is unclear, do not default to:

- centered SaaS hero
- generic gradient blob
- Inter on white with no stronger point of view
