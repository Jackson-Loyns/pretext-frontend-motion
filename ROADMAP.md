# Roadmap

This roadmap is intentionally short and public. It exists to help users and contributors understand what matters next.

## Current Focus

- make the installed skill more reliable across assistants
- improve real-project adoption through better integration examples
- improve discoverability on GitHub and npm

## Near Term

- add richer screenshots and motion previews to the repository
- tighten the installed skill so framework integrations are easier to reference
- improve starter quality and visual consistency
- publish clearer release notes for each npm and GitHub release

## Medium Term

- expand beyond bubbles to more reusable integration primitives
- add stronger framework-specific guidance for React, Vue, and Svelte
- add more public examples that map directly to real product surfaces

## Contribution Areas

Good contribution areas:

- documentation clarity
- install flow issues
- framework integration fixes
- starter polish
- example quality improvements
