# Changelog

All notable changes to this project should be recorded here.

The project follows a lightweight changelog style focused on user-visible changes, installer behavior, integration updates, and documentation improvements.

## v0.3.1

### Added

- Published npm installer package: `pretext-skill`
- Multi-CLI install flow for Codex, Claude Code, Cursor, Gemini CLI, GitHub Copilot, and more
- Official-family examples and blueprints for Pretext demo families
- Real TypeScript integrations for Vanilla, React, Vue, and Svelte
- Repository health files, CI, Dependabot, issue templates, and contributor guidance

### Changed

- README and docs reorganized into clearer install, usage, examples, and structure guides
- Installed bundle now includes integration references and framework adoption guidance

### Security

- Upgraded `@chenglou/pretext` to `0.0.5`
